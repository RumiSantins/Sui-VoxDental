# Indice de Documentacion Tecnica - SuiEgo

Bienvenido a la documentacion oficial del proyecto SuiEgo. Este repositorio contiene el codigo fuente y las configuraciones necesarias para la plataforma central SuiEgo y su extension especializada VoxDental.

La documentacion esta organizada de forma modular para facilitar su lectura y mantenimiento.

## Guia de Navegacion

### 1. Arquitectura de Sistemas
Vision general de la infraestructura, el stack tecnologico y el flujo de comunicacion entre los servicios de frontend, backend y el servidor de produccion.
[Ver Arquitectura](01_arquitectura.md)

### 2. Desarrollo Local
Instrucciones detalladas para configurar el entorno de trabajo en una maquina local, instalacion de dependencias y ejecucion en modo desarrollo.
[Ver Desarrollo Local](02_desarrollo_local.md)

### 3. Despliegue y Produccion
Manual para el mantenimiento del servidor en Google Cloud Platform, gestion de contenedores Docker, configuracion de Nginx y renovacion de certificados SSL.
[Ver Manual de Produccion](03_despliegue_produccion.md)

### 4. Referencia del Frontend (React)
Detalles sobre la estructura de componentes, gestion de estados globales, internacionalizacion y el sistema de diseño de SuiEgo.
[Ver Referencia Frontend](04_frontend.md)

### 5. Referencia del Backend (FastAPI)
Explicacion de la logica del servidor, integracion de motores de procesamiento de voz (Whisper/Vosk), modelos de datos y servicios auxiliares.
[Ver Referencia Backend](05_backend.md)
