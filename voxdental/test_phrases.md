# Guía de Frases y Protocolo - Sui VoxDental

Utiliza esta guía para conocer la mejor forma de hablarle al sistema y ver todos los términos que reconoce actualmente. A partir de la versión más reciente, el sistema utiliza un **Motor de Acrónimos Estrictos** para maximizar la velocidad y reducir a cero los errores fonéticos.

---

## 🎙️ Protocolo "Sui Voice" (Modo Acrónimo)

Para obtener la **mayor precisión**, debes dictar los comandos letreando el hallazgo y la cara, seguido del número de la pieza:
> **[CONDICIÓN (1 Letra)]** + **[SUPERFICIE (1 Letra)] (Opcional)** + **[NÚMERO DE DIENTE]**

*También puedes usar el formato decimal para los números: Por ejemplo, "dos punto uno" en lugar de "veintiuno".*

| Comando Tipo | Formato Acrónimo | Ejemplo Recomendado |
| :--- | :--- | :--- |
| **Simple (Letra + Cara + Diente)** | `C` + `O` + `16` | *"C O 16"* |
| **Resina con superficie** | `R` + `M` + `21` | *"R M 21"* o *"R M 2.1"* |
| **Diente Completo (Sin Cara)** | `CR` + `11` | *"C R 11"* |
| **Ausencias** | `X` + `18` | *"X 18"* |
| **Múltiple** | `C`+`O`+`16` y `R`+`M`+`21` | *"C O 16 y R M 21"* |
| **Corrección** | `B` + `16` | *"B O 16"* (Borra solo Oclusal) o *"B 16"* (Total) |
| **Finalizar** | --- | *"Sui fuera"* / *"Parar Sui"* |

---

## 1. Letras Oficiales de Patologías y Tratamientos

- **Caries (C)**: *"C O 16"*, *"C M 11"*.
- **Resina (R)**: *"R M 21"*, *"R V 22"*.
- **Amalgama (A)**: *"A D 46"*, *"A O 15"*.
- **Corona (CR)**: *"C R 11"*. (Dibuja un borde dorado grueso)
- **Endodoncia (E)**: *"E 12"*, *"E 44"*. (Colorea las raíces)
- **Extraer (EX)**: *"E X 48"*, *"E X 14"*. (Colorea las raíces)

## 2. Letras Oficiales de Superficies (Caras)

- **Oclusal / Incisal**: `O` / `I`
- **Mesial**: `M`
- **Distal**: `D`
- **Vestibular**: `V`
- **Palatina / Lingual**: `P` / `L`

## 3. Dientes Ausentes (X Azul)
- *"X 18"*, *"X 36"*, *"X 15"*.

## 4. Comandos de Corrección (Letra B)
- **Borrado Total**: *"B 11"* (Limpia totalmente el diente).
- **Borrado Selectivo**: *"B O 16"* (Quita solo el dibujo Oclusal de la 16 y mantiene el resto).

## 5. Finalizar Sesión (Hands-Free Stop)
El comando de parada sigue soportando lenguaje natural natural:
- **"Sui fuera"**, **"Chao Sui"**, **"Parar Sui"**, **"Shui fuera"**.

## 💡 Tip Pro: El Modo Híbrido (Acercamiento + Voz)
Puedes reducir aún más lo que tienes que hablar abriendo el Panel Manual:
1. Haz **clic** en un diente en el Odontograma (pej. Pieza 15) para abrir su ventana.
2. Mientras la ventana esté abierta, el motor ya sabe en qué diente estás. No necesitas decir el número.
3. Solo di: **"R M"** o **"C O"** o **"E X"**.
4. ¡El sistema aplicará el comando instantáneamente a la pieza que tienes seleccionada en pantalla!
