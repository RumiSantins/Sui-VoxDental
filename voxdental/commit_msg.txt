feat(core): Intelligent Voice Odontogram System, NLP Analytics & Clinical Optimizations

This commit consolidates the week's work focused on elevating VoxDental
from a baseline platform to a world-class autonomous clinical assistant.
We have implemented comprehensive voice control, an active learning 
(feedback) system, clinic-optimized multimedia support, and modular 
authentication flows.

🌟 Key Features:
- feat(voice): Dual voice engine integration (Vosk/Whisper) with Profile selector.
- feat(voice): Tactile AI Feedback interface (Thumbs Up/Down) inside the odontogram.
- feat(voice): "Assisted Correction" pop-up panel for retraining failed NLP commands.
- feat(admin): New secret "Voice Intelligence Panel" to analyze NLP hits and misses.
- feat(media): Evidence upload system (images/videos) attached to individual teeth.
- feat(media): Ultra-fast video remuxing using FFmpeg (preserving raw clinical quality without lag).
- feat(media): Automated, asynchronous thumbnail generation for clinical videos.
- feat(auth): Native Google Login integration with automatic account creation.
- feat(ui): Redesigned "Clinical Summary Panel" featuring an ultra-professional, Print-Ready CSS layout.
- feat(ui): Settings toggle to Enable/Disable action audio feedback (sounds) during tooth annotations.
- feat(ui): Customizable avatars and visual medical specialty selection in the profile.

🛠️ Bug Fixes & Refactors:
- fix(state): Resolved state desynchronization (odontogram fully resets upon changing patients).
- fix(auth): Password change section intelligently hides if the user logged in via social auth (Google).
- fix(admin): Direct access to the Analytics Panel from the login screen without corrupting the token tree.
- refactor(db): Refactored SQLAlchemy schemas (`SpeechReport`, `ToothMedia`) and fixed cascading on user factory reset.

🌐 Architecture & Security:
- JWT (JSON Web Tokens) implementation injecting passive provider states (`is_google` flag).
- Refinement of concurrent promise architecture for asynchronous API interactions.
